# E-Commerce - Front End Mentor

Esse projeto teve como foco eu exercitar os meus conhecimentos em relação ao typescript 
focando no front-end, ademais, como era um projeto que eu já tinha feito em javascript
puro de forma procedural decidi aprofeitar e praticar em POO. 
Não me foquei em seguir os principios SOLID pelo fato de ser literalmente o primeiro 
projeto usando esse paradigma queria apenas curtir o desenvolvimento, e bom, foi bem
divertido.

## Direitos

Reservo todos os direitos de design e criativo ao site [Front End Mentor](https://www.frontendmentor.io/)
que foi basicamente responsavel por disponibilizar o projeto a nos desenvolvedores iniciantes. Para ir para o desafio [clica aqui](https://www.frontendmentor.io/challenges/ecommerce-product-page-UPsZ9MJp6).
